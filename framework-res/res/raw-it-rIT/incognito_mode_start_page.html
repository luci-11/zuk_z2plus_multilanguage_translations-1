Sei in modalità Incognito. Quando chiudi una finestra Incognito, tutte le pagine che hai visitato non appariranno nella cronologia del browser o nella cronologia ricerca e non lascerai nessuna traccia (come i cookie). Tuttavia tutti i file scaricati e i segnalibri creati saranno preservati.

Utilizzare la modalità Incognito non influenza persone, server o software. Fai attenzione a：

  • Siti web che collezionano o condividono informazioni su dite
  • ISP o impiegati che tracciano le pagine che visiti
  • Software malevoli che tracciano ciò che digiti in cambio di smiley gratuiti
  • Sorveglianza di agenti segreti
  • Gente alle tue spalle
